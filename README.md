# 📖 About The Project

The **Smart Hotel Booking & Recommendation System** is an AI-powered desktop application that enables users to search, book, and receive personalized hotel recommendations based on their preferences.

Unlike a traditional booking application, this system integrates **Machine Learning** to recommend the most suitable hotels using a **Content-Based Recommendation Engine**.

The recommendation engine analyzes user-selected facilities and hotel attributes to provide intelligent suggestions using **Cosine Similarity** and **MultiLabelBinarizer** from **Scikit-learn**.

This project demonstrates the practical application of Artificial Intelligence in travel and hospitality while showcasing desktop application development with Python.
